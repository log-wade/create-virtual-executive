# Cycle 8 — D1 Phone MVP (Twilio Gather + complete API)

## Summary

- **`POST /api/chat/complete`** — JSON `{ personaId, messages }` → `{ text }` using `completeExecutiveReply` (voice workers / tools; web keeps streaming `/api/chat`).
- **`POST /api/twilio/voice`** — Incoming-call TwiML: `<Gather input="speech">` → `POST /api/twilio/voice/gather` (absolute URL from `PUBLIC_BASE_URL` or request origin).
- **`POST /api/twilio/voice/gather`** — Reads `SpeechResult`, calls `completeExecutiveReply` with a single user turn, responds with `<Say>` + optional follow-up gather (“Anything else?”).
- **`lib/twilio/twiml.ts`** — XML escape, say truncation, `getPublicBaseUrl`, `twimlResponse`.
- **Middleware** — Skip Upstash rate limit for `/api/twilio/*`.
- **`voice-worker/`** — `package.json` + `server.mjs` WebSocket stub + README (Media Streams TBD); not required for Gather MVP.

## Env

- `PUBLIC_BASE_URL` — recommended in production for correct Gather `action` URLs.
- `TWILIO_VOICE_PERSONA_ID` — required for useful TwiML; missing → spoken config hint.
